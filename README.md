# Advent of Code 2024

Solutions for [Advent of Code](https://adventofcode.com/).

I'm doing these in rust because I want to get better at rust.

### New Day Setup

1. `cargo new dayNN --vcs none`.
2. Add `src/bin/part1.rs` and `src/bin/part2.rs`.
3. Delete `main.rs`.

Run with `cargo run --bin part1` or `cargo run --bin part2`.
